# 10 訪問優先順位ボード

今週どの顧客を優先訪問すべきかを、理由つきで並べる営業支援ツール。長期未訪問、契約更新接近、失注兆候、大型案件、役員商談予定などをスコア化し、訪問理由の内訳まで確認できる。

- 起動: `cd 10_visit_plan_builder` → `../.venv-linux/bin/python gui.py`
- 入力: `data/visits.csv`
- 出力: `results/weekly_visit_plan.csv`
- 見せ場: 上位訪問先の優先理由内訳を積み上げ横棒グラフで表示
